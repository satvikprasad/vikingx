# VikingX
## Instructions
### Quickstart
To run:
`make run`
To build:
`make build`

.env must contain:
```
WEBHOOK_PHRASE=
ACCESS_KEY=
SECRET_KEY=
PASSPHRASE=
PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
```

Have fun.
